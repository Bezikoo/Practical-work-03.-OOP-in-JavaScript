# Survival Inventory System (Resident Evil Style)

Цей проєкт є реалізацією ігрового інвентаря в стилі survival-horror (натхненний серією Resident Evil) з використанням об'єктно-орієнтованого підходу в JavaScript.

Проєкт виконано в межах **Практичної роботи №3** з дисципліни "Web-технології та Web-дизайн".

## 🎮 Про проєкт

Система імітує складний менеджмент ресурсів, де гравець має не просто збирати предмети, а стратегічно керувати обмеженим простором у 2D-сітці.

### Основні можливості:
- **Система сітки (Grid System)**: Інвентар розміром 8x10 клітинок. Кожен предмет має власну геометрію (1x1, 1x2, 2x2 тощо).
- **Розумне розміщення**: Алгоритм автоматичного пошуку вільного місця при додаванні предметів.
- **Дефрагментація (Сортування)**: Функція автоматичного перепакування предметів для максимальної оптимізації простору.
- **Система крафту та апгрейдів**: Можливість комбінувати ресурси (наприклад, трави) або покращувати зброю за допомогою модифікаторів.
- **Реактивний UI**: Інтерфейс автоматично оновлюється при будь-якій зміні в моделі даних (патерн Observer).
- **Dark Theme**: Атмосферний дизайн у стилі survival-horror.

## 🛠 Технологічний стек
- **Vanilla JavaScript (ES6+)**: Використання класів, приватних полів (`#`), модулів.
- **HTML5 & CSS3**: Гнучка сітка (CSS Grid) та сучасна верстка без сторонніх бібліотек.
- **OOP & SOLID**: Суворе дотримання принципів чистої архітектури.

## 🚀 Як запустити проєкт

Оскільки проєкт використовує **JavaScript модулі (ES Modules)**, його не можна запустити просто відкривши файл `index.html` подвійним кліком (через політику безпеки CORS).

1. Встановіть розширення **Live Server** у VS Code або інший локальний сервер.
2. Відкрийте папку проєкту.
3. Запустіть сервер та відкрийте `index.html` через нього.

## 📁 Структура проєкту
- `DOMAIN_MODEL.md` — Аналіз механік та UML-діаграма.
- `index.html` — Структура інтерфейсу.
- `styles.css` — Стилі інтерфейсу.
- `js/app.js` — Точка входу та контролер.
- `js/core/` — Ядро логіки (Інвентар, Крафт).
- `js/models/` — Моделі даних предметів.
- `js/ui/` — Логіка відображення.
